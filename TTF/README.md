用于英文文献格式转换的插件，适用于谷歌浏览器，转换格式可直接导入谷歌翻译。

插件添加方法：
       1、打开谷歌浏览器；
       2、搜索栏输入：chrome://extensions/
       3、右上角——打开开发者模式；
       4、点击左上角——加载已解压的扩展程序，选择本TTF文件夹，确定；
       5、浏览器右上角出现插件图标。

Do按钮：文字格式转换
Go按钮：将文字push到谷歌翻译文本框中
Dogo按钮：Do + Go
